# E-Commerce-Sales-Analysis

## Project Overview
Analyzed e-commerce sales data to identify sales trends, top-performing products, and revenue insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Top selling products identified
- Revenue trends analyzed
- Category-wise sales comparison
- Business recommendations provided

## Files
- ecommerce_analysis.ipynb
- dataset.csv
- requirements.txt

## Conclusion
This analysis helps businesses understand customer purchasing behavior and improve sales strategies.
